Overview

Because there are so many functions, there is a need to classify functions based on their behaviors or classifications. This section discusses the various types and describes how to differentiate them. It also provides the corresponding functions for each type of function.

Function Classes

Classifications take on two different forms: those that describe the behavior, and those that describe the characteristics of a function.

Behavior-based classifications describe the overall behavior of the function; that is, what are the particular features that describe its overall shape over different inputs?

Characteristic-based classifications describe particular typically-localized aspects of the function, such as the functions maxima or minima, intercepts, continuity, etc.

Behavior-based classifications are discussed across the next few chapters, while characteristic-based classifications are discussed in the next sections.

Describing a Function's Input or Output

Intercepts

One such way to describe a function is to identify where the function intersects with the xxx or yyy axes.

This is a particularly useful tool when graphing a function, because it gives a clear starting point for plotting.

For a 2D function (that is, only one input, and thus one corresponding output) there are 2 intercepts.

The x−intercept\textcolor{cyan}{x-\textit{intercept}}x−intercept is the point at which the function intersects with the xxx axis. That is, what value of xxx results in a value of the function of 000?

You can find the xxx-intercept by solving for xxx when f(x)=0f(x) = 0f(x)=0.

The other intercept is the y−intercept\textcolor{cyan}{y-\textit{intercept}}y−intercept, which is the point at which the function intersects with the yyy axis. That is, what value of yyy results from an input value of x=0x=0x=0?

You can find the yyy-intercept by calculating f(0)f(0)f(0).

Some functions may have more than one xxx-intercept, but no function should have more than one yyy-intercept (formula and equations might, however!). Ultimately, this depends on the nature of the function itself, and will be described in more detail upon those details.

Continuity

Functions may be described as continuous or discontinuous.

Functions lacking abrupt changes in value are considered continuous\textcolor{cyan}{\textit{continuous}}continuous.

The "simplest" kind of discontinuity is a Removable Discontinuity\textcolor{cyan}{\textit{Removable Discontinuity}}Removable Discontinuity, which is one in which the function approaches the same values regardless of the direction xxx is approaching.

A hollow circle denotes that the value does NOT exist at this location. It indicates a hole\textcolor{cyan}{\textit{hole}}hole in the graph.

A solid circle denotes an output at a particular point, which should be labelled. In the case of the Figure above, it is labelled at x0x_0x0​

Jump Discontinuity

Another type of discontinuity is a Jump Discontinuity\textcolor{cyan}{\textit{Jump Discontinuity}}Jump Discontinuity, where the function approaches different values based on the direction xxx is approaching.


Figure 1.3.2: Jump Discontinuity at x0

Bounded Functions

A bounded\textcolor{cyan}{\textit{bounded}}bounded function is one that is finite.

In other words, all values of the function will lie between some values kkk and mmm, where both kkk and mmm are real numbers. That is, the following will be true:

f(x)≤kf(x)≥m\begin{aligned} &f(x) \leq k \\ &f(x) \geq m \\ \end{aligned} ​f(x)≤kf(x)≥m​

Because both kkk and mmm are real numbers, this enforces the concept that no value of f(x)f(x)f(x) will be infinite!

Unbounded\textcolor{cyan}{\textit{Unbounded}}Unbounded functions are those that extend toward infinity. Many functions are unbounded. For instance, Linear functions are unbounded.

Some functions are "semi-bounded", meaning they are only bounded on one side of the output. That is, there may be functions that are...

  • Bounded above\textcolor{cyan}{\textit{Bounded above}}Bounded above, which means that only f(x)≤kf(x) \leq kf(x)≤k holds true. In other words, all function values are less than (or equal to) some real number kkk.
  • Bounded below\textcolor{cyan}{\textit{Bounded below}}Bounded below, which means that only f(x)≥mf(x) \geq mf(x)≥m holds true. In other words, all function values are greater than (or equal to) some real number mmm.

Smooth Functions

A smooth\textcolor{cyan}{\textit{smooth}}smooth function describes one such that the function is differentiable everywhere. Symmetry

A symmetrical\textcolor{cyan}{\textit{symmetrical}}symmetrical function describes a function such that the function is the same across an axis.

Typically, symmetry for functions applies across the yyy axis.

Functions like the Quadratic function is symmetric across the yyy axis. Some functions, like the Cubic function, are actually symmetric across f(x)=−xf(x)=-xf(x)=−x (that is, reflected diagonally).
